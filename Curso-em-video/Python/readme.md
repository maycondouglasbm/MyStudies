## Tipos primitivos

- INT --> 7, -4, 0, 9875
- FLOAT --> 4.5, 0.076, -15.223, 7.0
- BOOL --> True, False
- STR --> 'Olá', '7.5', 'terra' 

## Operadores artméticos

- (+) adição
- (-) subtração
- (*) multiplicação
- (/) divisão 
- (**) potência
- (//) divisão inteira
- (%) resto da divisão
- (n**(1/2)) raiz quadrada ou pow(n, (1/2))

## Ordem de precendência

- 1ª) ()
- 2ª) **
- 3ª) *, /, //, %
- 4ª) +, -

- \n --> quebra linha

- != --> diferente
